# Figma Make Prompt: Transfer Portal Aggregator Website UI

Design a modern, clean, sports-analytics style website UI for a
**College Sports Transfer Portal Aggregator Platform**.

The platform aggregates transfer portal data for college teams and
players, presenting it in a simple, beginner-friendly format.

The design should feel: - Professional and data-driven (like ESPN + The
Athletic + modern SaaS dashboard) - Clean and readable - Slightly bold
sports aesthetic (strong typography, confident spacing) - Minimal
clutter - Easy for non-experts to understand

Use: - Neutral light background (off-white or light gray) - Dark navy or
charcoal primary color - Accent color in electric blue or subtle neon
green - Clear typography hierarchy (bold headlines, clean sans-serif
body text) - Card-based UI layout - Rounded corners (8--12px) - Subtle
shadows for elevation - Responsive desktop-first design

# Overall Feel

The website should feel like: - A professional sports analytics
dashboard - Easy for beginners - Informative for hardcore fans -
Data-rich but not overwhelming - Modern SaaS aesthetic with sports
identity

Avoid: - Clutter - Too many colors - Heavy gradients - Overly complex
data visuals

Focus on: - Clarity - Scanability - Clear hierarchy - Strong
typography - Well-structured card layouts
